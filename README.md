# Sistema de Bonos de Sueldo Digital

Sistema para la creación y firma digital de bonos de sueldo.

## Estructura del proyecto
- `/backend` - Servidor y API
- `/frontend` - Interfaz de usuario
- `/docs` - Documentación

## Tecnologías
- Backend: Node.js, Express, MongoDB
- Frontend: HTML, CSS, JavaScript
- PDF: PDFKit
- Firma digital: Crypto
